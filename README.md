## 🧾 Featured Project: Order Flow  
*A system designed to prevent overselling and inconsistent stock under concurrent operations.*

* **Concurrency Control:** Row-level locking during order creation to eliminate race conditions  
* **Data Integrity:** Enforced SKU uniqueness and relational constraints at the DB level  
* **Efficient Backend Logic:** Transactional RPCs to handle order processing and aggregation
